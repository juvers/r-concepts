# Press List Item

## NOTE: WIP - this is mostly a copy of PODCAST, including these docs

The Podcast component is used in a few pages on RC, so it is a generic component
to allow usage in multiple places.
There are a lot of props needed to create the component and most currently come directly from
sanity.
In addition to sanity data, we also need to get the logo images that create the links to the
podcast sources. These currently come from our local file system.

To see an example of data validation, look at
`/rockefellercenter/src/blocks/EventPodcastBlock.tsx`
