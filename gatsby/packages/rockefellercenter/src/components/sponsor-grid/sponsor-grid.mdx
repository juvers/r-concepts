# Sponsor Grid

This component allows for creating a reusable grid to display sponsor logos

The main difference between primary and secondary logos is the sizing of
each logo

Primary = two across + larger logos

Secondary = three across + smaller logos

### Props

<Props of={SponsorGrid} />

### Usage

### Primary Sponsor Grid

<Playground>
  <SponsorGrid
    title="Primary Sponsors"
    isPrimary={true}
    logos={[
      {
        fluid: getMockFluidImage({width: 240, height: 170}).childImageSharp
          .fluid,
        alt: 'first primary sponsor',
      },
      {
        fluid: getMockFluidImage({width: 240, height: 170}).childImageSharp
          .fluid,
        alt: 'second primary sponsor',
      },
    ]}
  />
</Playground>
