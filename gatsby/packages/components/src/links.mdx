# Intro

Box components are only there to make the background dark

Both links are variants of the Link component provided by theme-ui

using variant "underline", "arrow" or "button" will add the proper styling to the link

## Properties

<Props of={Link} />

```ts
 import {Link} from '@tishman/components';

<Link variant='underline' href='#'>Buy Tickets</Link>

<Link variant='arrow' href='#'>Contact Us<ArrowSvg /></Link>

<Link variant='button' href='#'>Plan an Event</Link>

<Link variant='card' href='#'>Plan an Event</Link>

<Link variant='footer' href='#'>Footer Link</Link>
```
