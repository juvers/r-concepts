# TabMenu

TabMenu takes a list of tab objects and displays them in a scrollable list with an active state animation. It takes a callback funtion and passes in the index of the active tab.

### Properties

<Props of={TabMenu} />

## Basic usage

```ts
import {TabMenu} from '@tishman/components';

<TabMenu />;
```

### Playground

<Playground>
