# IntrinsicBox

A ThemeUI [Box], but with a `ratio` prop like ThemeUI [AspectRatio].

The differences between this component and [AspectRatio] are:

- [AspectRatio] hides overflow (and thus requires content to be positioned),
  while this component shows overflow, and content is allowed to flow.

- [AspectRatio] only supports one `ratio` value, while this component supports
  multiple for responsive designs

- It supports `minWidth` and `maxWidth` props, which can also be responsive

It also maintains the ratio differently, by sandwiching content
between floated `::before` and cleared `::after` [pseudo elements]
rather than [padding and positioning].

> To use [Flex] instead of [Box], see [IntrinsicFlex].

> For displaying images with intrinsic ratio, see [IntrinsicImage].

## Basic usage

```ts
import {IntrinsicBox} from '@tishman/components';
```

### With `ratio`:

<Playground>
  <IntrinsicBox
    ratio={16 / 9}
    sx={{border: '1px solid', borderColor: 'accentSecondary'}}
  >
    <Heading>Intrinsic Box</Heading>
    <Text>This box keeps its intrinsic ratio when resizing</Text>
    <Text>But it will allow overflow to be visible</Text>
  </IntrinsicBox>
</Playground>

### With `minWidth` and `maxWidth`:

<Playground>
  <IntrinsicBox
    ratio={16 / 9}
    minWidth={350}
    maxWidth={500}
    sx={{border: '1px solid', borderColor: 'accentSecondary'}}
  >
    <Heading>Intrinsic Box</Heading>
    <Text>This box keeps its intrinsic ratio when resizing</Text>
    <Text>But it will allow overflow to be visible</Text>
  </IntrinsicBox>
</Playground>

### With [Box] properties:

<Playground>
  <IntrinsicBox
    ratio={16 / 9}
    p={4}
    color="primary"
    bg="accent"
    sx={{
      border: '1px solid',
      borderColor: 'currentColor',
    }}
  >
    <Heading>Intrinsic Box</Heading>
    <Text>This box keeps its intrinsic ratio when resizing</Text>
    <Text>But it will allow overflow to be visible</Text>
  </IntrinsicBox>
</Playground>

### With responsive ratio

<Playground>
  <IntrinsicBox
    ratio={[1, 4 / 3, 16 / 9]}
