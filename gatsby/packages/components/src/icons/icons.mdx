# Icons

A number of icons are provided as React components for use in Tishman projects.

They have the following features:

- Default reasonable width and height, which can be overridden
  with props/styles.
- Monotone stroke and fill colors using [currentColor], so the icon color
  can be changed just by setting the 'color' style on the icon or an ancestor.

If you need to add an icon, see [Adding Icons].

## Usage

To use an icon, you can just import and render it:

```tsx
import {ArrowSvg} from '@tishman/components';
```

</Playground>

## Adding Icons

> Note: Be sure to export your icon from [icons/index.ts]!

Any icons in the [icons directory] will be automatically converted to
React components. Note the following assumptions and constraints apply:

- The svgs should be monotone, with negative space being transparent.
- The color of the icons will not be preserved;
  stroke and fill colors will be replaced with [currentColor].
- Icons will be automatically sized with default width and height to match
  their [viewBox].

See the [gatsby config] for the [gatsby-plugin-react-svg] for more.
