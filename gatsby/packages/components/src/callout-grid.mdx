### With Title Block

Some of the designs have a grid with a title and caption block.

To add a title block to the grid, you must pass title and caption props,

directly into the CalloutGrid.

Nice thing about the TitleBlock is we will use javascript to calculate

the offset so no matter the content, it will calulate the offset automatically.

### Unique layout (holiday lp example)

Lastly some of our layouts are more custom with different width cards and different grid styling

We can change the offset of the grid by adding an offset prop `offset={360}`

as well as extra styling on the CalloutGrid to achieve the layout `sx={{mb: [0, null, 150]`

Lastly we also need to apply custom styling to each CalloutGridCard to acheive the unique layout

See [Tall Narrow Callout Grid Card](/components/callout-grid-cards#tall-narrow-card-with-extra-margin)

See [Wide Callout Grid Card](/components/callout-grid-cards#wide-card-with-no-margin-on-desktop)

for examples of different custom card styling
