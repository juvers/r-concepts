### With custom styling

#### Tall narrow card with extra margin

Some designs have custom grid cards that break out of the default layout.

we can manipulate Callout Grid Cards in a few ways

adding sx object to overwrite default styling. `sx={{ml: 7}}`

we can also overwrite default maxWidth and ratio to use different image styling

`ratio={303 / 529}`

`maxWidth={303}`

See [Callout Grid with unique layout](/components/callout-grid#unique-layout-holiday-lp-example)

<Playground>
  <CalloutGridCard
    index={1}
    sx={{ml: 7}}
    ratio={303 / 529}
    maxWidth={303}
    fluid={getMockFluidImage({width: 303, height: 529}).childImageSharp.fluid}
    title={'Maps & Directions'}
    caption={
      'Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.'
    }
    links={[{url: '#', label: 'learn more'}]}
  />
</Playground>
