# Text Variants

### Example using ThemeUi Heading component

(Primitive heading component, defaults to `h2`.)

```ts
import {Heading, Styled} from 'theme-ui';

<Heading variant="heroTitle">heroTitle</Heading>;

<Styled.h1>Using Styled syntax</Styled.h1>;
<Heading variant="styles.h1">Using styles.h1 syntax</Heading>;
```

### Example using ThemeUi Text component

(Primitive typographic component.)

```ts
import {Text} from 'theme-ui';

<Text variant="mediumP">mediumP</Text>;
```

### Example using ThemeUi Styled component

(Styled.h1 to Styled.h5 + Styled.p)

```ts
import {Styled} from 'theme-ui';

<Styled.h1>Using Styled syntax</Styled.h1>;
```
