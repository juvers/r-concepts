# Buttons

Buttons are built using the [ThemeUI Button] component and variants.
In most cases, the Button component should be used with a [Button variant].
In cases where a button needs specific design elements (for example,
the [MenuToggleButton]), The button component should be built on top of the
Button component.

## Button

This is the [ThemeUI Button] component.
