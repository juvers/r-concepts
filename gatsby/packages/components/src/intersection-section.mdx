# IntersectionSection

`IntersectionSection` is a special case of [Section] that
calls an `onIntersection` callback whenever the section element intersects
the root element.

> How does this differ from [AnchorSection]? The main difference is that
> `AnchorSection` requires an `id`, and **binds to the location hash**
> such that the hash updates when the section intersects the root,
> and the section is scrolled into view when the location hash changes.
