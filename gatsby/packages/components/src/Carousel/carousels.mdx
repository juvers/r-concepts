## Creating a Carousel

A carousel consists of two building blocks: a [CarouselContextProvider]
and the [useCarouselContext] hook. In many cases, a carousel can be composed
from the existing [CarouselList] and [CarouselNavigation] components.

For example, here is the implementation of [Carousel]:

```jsx
const Carousel = (props) => (
  <CarouselContextProvider>
    <CarouselList {...props} />
  </CarouselContextProvider>
);
```

### CarouselContextProvider

Provides a [Pagination] object (from [usePagination]) as context.

### useCarouselContext

A hook for the component parts of a carousel. receives a [Pagination]
object from the nearest [CarouselContextProvider].

<Playground>
  {() => {
    const color = ['text', 'primary', 'secondary', 'accent', 'muted'];
    const Item = ({index}) => {
      const {page, pages} = useCarouselContext();
      const delta = index - page;
      return (
        <Box py={9} bg={color[index % color.length]} sx={{textAlign: 'center'}}>
          <Text>{`page ${index + 1} of ${pages}`}</Text>
          <Text>
            {delta
              ? `${delta} from ${delta > 0 ? 'right' : 'left'}`
              : 'active!'}
          </Text>
        </Box>
      );
    };
    return (
      <CarouselContextProvider>
        <CarouselNavigation />
        <CarouselList
          __debug
          color="background"
          gap={0}
          autoSize="max(20%, 180px)"
          maxWidth="max(20%, 180px)"
        >
          {Array.from(Array(10).keys(), (index) => (
            <Item key={index} index={index} />
          ))}
        </CarouselList>
      </CarouselContextProvider>
    );
  }}
</Playground>

### CarouselList

A [CSS Scroll Snap] implementation of a carousel.

Receives the same props as [Carousel].

Requires a [CarouselContextProvider] ancestor.

### CarouselNavigation

A simple navigation component with next and previous buttons.

Requires a [CarouselContextProvider] ancestor.

<Playground>
  {() => {
    return (
      <CarouselContextProvider>
        <CarouselNavigation />
        <CarouselList color="background" gap={0} autoSize="100%">
          <Text p={9} bg="primary" children="item 1" />
          <Text p={9} bg="secondary" children="item 2" />
          <Text p={9} bg="muted" children="item 3" />
          <Text p={9} bg="accent" children="item 4" />
          <Text p={9} bg="accentSecondary" children="item 5" />
        </CarouselList>
      </CarouselContextProvider>
    );
  }}
</Playground>
