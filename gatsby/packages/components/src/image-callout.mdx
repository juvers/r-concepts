# Image Callout

Image Callout is used throughout the site.

Desktop design is left aligned image with border
and title, caption, link block on the right side.

On mobile, the image and text blocks stack and
border is removed from the image.

All props are required to render

```ts
import {ImageCallout} from '@tishman/components';
```
